# De opdracht
We hebben de opdracht gekregen om met een multidisciplinair team van SD RV EN MV een concept voor speelse immersieve ruimte van 2,5 meter bij 2,5 meter. De bedoeling hiervoor is het maken van een interactieve installatie met een mapped projectie die ge√Ønspireerd is op het werk van de Nederlandse kunstenaar die wij hebben gekozen(Daan Rosegaarde). De installatie is ruimtelijk en immersief, interactief, hedendaags en spannend maar veilig. Alle ruimtes samen vormen een kunstspeeltuin, dus moeten wij goed nadenken over de ruimtelijk indeling.
